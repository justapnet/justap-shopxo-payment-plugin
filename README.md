# shopxo 支付插件

## 简介
适用于 [shopxo](https://www.shpox.net) 商城的开源聚合支付插件。

本插件包含多个插件以适配多个支付渠道。

## 安装

### 从 shopxo 官方应用商店安装

1. 登录 shopxo 应用商店
2. 免费购买  https://store.shopxo.net/goods-181.html 
3. 安装即可


### 从 开源聚合官方下载安装

1. 从本仓库 release 下面下载最新插件包
2. 登录 shopxo 后台，上传插件包安装并使用
